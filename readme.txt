== Frequently Asked Questions ==

= Can I customize the appearance of the recommendations? =
Yes, you can! SparkAIR provides various customization options, such as choosing the number of products to display, adjusting the appearance and configuring the design to match your store\'s.

= Is SparkAIR compatible with my WooCommerce theme? =
SparkAIR is designed to be compatible with most WooCommerce themes. However, if you encounter any compatibility issues, our support team is here to assist you in resolving them.

= Does SparkAIR work with variable products? =
Yes, the plugin is fully compatible with variable products. It will display the recently viewed variations of a product if the customer has viewed them.

= Will SparkAIR slow down my website? =
No, SparkAIR is optimized for performance and will not significantly impact your website\'s loading speed. We\'ve ensured that it operates efficiently to provide a seamless shopping experience.

= Does the free version of the plugin have limitations compared to the PRO version? =
Yes, the free version comes with basic functionality. The PRO version offers advanced features like analytics on conversion and clicks and showing the products in a carousel. Check [our website](https://www.sparkplugins.com/sparkair) for more info.

= How do I upgrade from the free version to the PRO version? =
Upgrading is easy! Simply purchase the PRO version from [our website](https://www.sparkplugins.com/sparkair) and follow the installation instructions provided. Your data from the free version will be seamlessly transferred.

Sure we can! Extended FAQ can be found on the [SparkPlugins website](https://www.sparkplugins.com/sparkair). If your question is not listed over there either, please do not hesitate to contact us via [support](https://www.sparkplugins.com/support).

Automatic installation is the easiest option -- WordPress will handle the file transfer, and you won‚Äôt need to leave your web browser. To do an automatic install of **SparkAIR**, log in to your WordPress dashboard, navigate to the Plugins menu, and click "Add New."
 
In the search field type "SparkAIR" then click "Search Plugins." Once you‚Äôve found us, you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by Clicking "Install Now" and WordPress will take it from there.

= Manual installation =

Manual installation method requires downloading the **SparkAIR** plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).

= Updating =

Automatic updates should work smoothly, but we still recommend you back up your site.
